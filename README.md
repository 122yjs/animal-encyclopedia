# 동물도감 54

초등학교 3학년 과학 「동물의 생활」 수업용 게임형 동물도감입니다.

## 바로 사용하기

- 기본 도감: https://122yjs.github.io/animal-encyclopedia/
- 질문방 없는 버전: https://122yjs.github.io/animal-encyclopedia/no-question.html
- 사진 출처: 위키미디어 커먼즈 등 (각 동물 상세창 표기)

## 교사용 질문방 연결 방법

1. 기본 도감 페이지 접속
2. `질문방 설정` 클릭
3. MagicSchool 학생용 참여 링크 붙여넣기
4. 생성된 수업용 링크 또는 QR을 학생에게 공유

## 주의사항

- 학생에게 기본 GitHub Pages 주소를 그대로 QR로 공유하지 않습니다. (해당 교사의 질문방이 연결되지 않습니다)
- AI 질문방을 사용하지 않을 경우 처음부터 `질문방 없는 버전(no-question.html)`을 학생에게 공유합니다.
