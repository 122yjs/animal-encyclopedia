# 🌲 숲 테마 동물도감 리디자인 (V2) 변경 사항

이 문서는 `main` 브랜치 대비 이번 `feat/ui-redesign-v2` 브랜치에서 새롭게 변경 및 적용된 사항들을 요약한 것입니다.

## 1. 개요 및 목적
기존의 다소 밋밋했던 교육용 동물도감 UI를 학생들이 몰입할 수 있도록 **RPG 게임 및 숲 탐험(Gamified Forest) 테마**로 전면 개편했습니다. "도감 마스터 미션"이라는 게임성을 시각적으로 강조하기 위해 깊이 있는 우드 톤과 골드 포인트, 그리고 최신 글래스모피즘(Glassmorphism) 스타일을 적용했습니다.

---

## 2. 기술적 변경 사항
* **Tailwind CSS v4 도입**: 별도의 빌드 도구 설정 없이 순수 정적 사이트(Vanilla HTML) 환경을 유지하기 위해 `@tailwindcss/browser` CDN을 `index.html`과 `no-question.html`에 일괄 적용했습니다.
* **빌드 스크립트 수정 (`scripts/build.js`)**: 새로 추가된 숲 테마 이미지 에셋들(`assets` 폴더)이 배포본(`dist/` 폴더)에 자동으로 복사되도록 빌드 파이프라인을 개선했습니다.
* **의존성 유지**: 기존 프로젝트의 주요 규칙("zero-npm-dependencies")을 준수하여 추가 패키지 설치 없이 UI만을 업그레이드했습니다.

---

## 3. 에셋(Assets) 추가
새로운 디자인 테마를 위해 AI를 활용하여 4종의 고품질 스프라이트 이미지를 생성하고 적용했습니다.
* `assets/sprites/owl-mascot.png`: 화면 좌측 상단에서 학생들을 안내하는 부엉이 마스코트 (인사, 탐색 등 포즈)
* `assets/sprites/forest-bg.png`: 지브리 애니메이션 스타일의 따뜻한 숲 파노라마 배경 이미지
* `assets/sprites/regions.png`: 5가지 지역(우리 주변, 숲, 강/호수, 바다, 특별환경)을 상징하는 배지 이미지
* `assets/sprites/game-icons.png`: 미션 완료 시 부여되는 보상 및 아이콘 요소

---

## 4. UI / UX 스타일 개편
`styles.css`의 CSS 변수(`:root`)를 전면 교체하여 전체 시스템의 테마를 변경했습니다.

### 🎨 색상 및 질감 (Color & Texture)
* **배경**: 단순한 단색에서 하늘 ➔ 풀밭 ➔ 황톳빛 흙으로 이어지는 **자연스러운 그라디언트 배경**으로 변경했습니다.
* **사이드바**: 밝은 톤에서 고급스러운 **다크 우드 패널(`rgba(45, 27, 14, 0.95)`)**로 변경하고, 가독성을 위해 텍스트를 골드와 아이보리 톤으로 맞췄습니다.

### 🃏 동물 카드 및 요소 (Glassmorphism)
* **카드 스타일**: 카드가 배경 위에 떠 있는 듯한 입체감을 주기 위해 **글래스모피즘 효과**(`backdrop-filter: blur()`)와 옅은 골드 글로우(`box-shadow`)를 추가했습니다.
* **애니메이션**: 마우스 호버(Hover) 시 카드가 가볍게 튀어 오르는 듯한 스프링 물리 효과를 주어 클릭을 유도했습니다.
* **버튼**: 기존의 단순 버튼을 보석 같은 **골드 그라디언트 버튼**으로 통일하여 게임 UI 느낌을 주었습니다.

### 🪟 모달 및 퀴즈 영역
* **오버레이**: 모든 모달(동물 상세 정보, 보상 화면 등)이 열릴 때 나타나는 뒷배경을 기존 어두운 회색에서 **반투명한 다크 브라운(`rgba(45,27,14,0.82)`)**으로 변경했습니다.
* **퀴즈 패널**: 퀴즈를 푸는 영역은 양피지 종이 느낌의 따뜻한 크림색(`var(--card-bg)`)으로 구성하여 글씨를 읽기 편하게 만들었습니다.

### 📌 사용성 개선 (진행바 및 스크롤)
* **커스텀 스크롤바**: 브라우저 기본 스크롤바 대신 테마에 맞춘 브라운/앰버 색상의 스크롤바를 적용했습니다.
* **글로벌 진행바(`sticky-progress`) 수정**: 
  * 화면 우측 상단에 표시되는 `🐾 0 / 54` 뱃지를 다크 우드 스타일로 변경했습니다.
  * **(중요 개선)** 퀴즈를 풀기 위해 모달을 열었을 때도, 학생이 도감 수집 진행 상황을 계속 볼 수 있도록 진행바의 `z-index`를 모달 오버레이보다 높게(`50`) 설정하여 화면에 항상 떠 있도록 수정했습니다.
* **퀴즈 모달 내 타이틀 고정**: 동물의 설명이 길어서 스크롤을 내리더라도, 현재 풀고 있는 문제의 번호(`문제 1 / 3`)가 화면 밖으로 벗어나지 않도록 타이틀 영역에 `position: sticky`를 적용했습니다.
