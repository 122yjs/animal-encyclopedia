# 수업 전 QA 체크리스트

실제 수업 환경에서 발생할 수 있는 문제를 사전에 확인하기 위한 점검표입니다.

## 1. 교사용 PC
- [ ] 기본 페이지 접속 확인 (https://122yjs.github.io/animal-encyclopedia/)
- [ ] `질문방 설정` 버튼 정상 클릭 확인
- [ ] MagicSchool 링크 붙여넣기 및 수업용 링크(QR) 정상 생성 확인
- [ ] 교실 내 전자칠판에서 전체 화면 전환 시 레이아웃 깨짐 없는지 확인

## 2. 학생용 태블릿 (Galaxy Tab A8 등)
- [ ] 교사가 제공한 학생용 QR로 접속 확인
- [ ] 메인 화면에서 `질문방 설정` 버튼 숨김 처리되었는지 확인
- [ ] 동물 카드 열기 정상 작동 및 텍스트 가독성 확인
- [ ] 퀴즈 풀기 기능 정상 작동 (오답 처리, 정답 처리)
- [ ] 도감 등록 기능 확인 (애니메이션, 도감 추가)
- [ ] 분류 게임 화면 진입 및 카드 이동, 되돌리기 조작 확인 (터치 인터페이스 문제 없는지)
- [ ] 수업 종료 후 설정 메뉴에서 `수업 기록 초기화`가 정상 동작하는지 확인

## 3. 질문방 없는 버전 (no-question.html)
- [ ] `no-question.html` 페이지로 직접 접속
- [ ] 메인 화면 및 설정 메뉴에서 `질문방 설정` 버튼이 일절 보이지 않는지 확인
- [ ] 퀴즈, 도감 보기, 분류 게임이 정상 작동하는지 확인
- [ ] 동물 상세 모달 내에 `더 궁금한 점 물어보기` 버튼이 존재하지 않는지 확인

## 4. 접근성 및 가독성
- [ ] 태블릿 밝기를 50%로 낮춘 상태에서도 텍스트 확인이 가능한지 (명도 대비)
- [ ] 터치 대상이 되는 버튼들의 크기가 충분히 넓게 확보되었는지
- [ ] 설정 메뉴에서 효과음 끄기가 정상 동작하는지
- [ ] 모든 모달(팝업) 창의 닫기 버튼이 명확히 보이고 동작하는지
