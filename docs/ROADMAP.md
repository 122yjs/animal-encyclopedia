# 동물도감 54 운영 개선 로드맵

## 1. 현재 상태 요약
초등학교 3학년 과학 「동물의 생활」 단원을 위한 게임형 동물도감 V2 구축이 거의 완료되었습니다. (숲 테마 UI 적용 완료, 온보딩 및 지역 미션 도입 완료)

## 2. 완료된 주요 개선
- 3단계 온보딩 튜토리얼 적용
- 지역별 마일스톤 및 보상(배지) 모달 구현
- 효과음 시스템 도입
- 사진 출처 일괄 표기 페이지 구성
- 로컬 이미지 매니페스트 적용
- `no-question.html` 빌드 시 자동 생성되도록 개선 (`scripts/generate-no-question.js` 스크립트)

## 3. 남은 핵심 과제

### P0. 수업 투입 전 필수 확인
- Galaxy Tab A8 실기기 QA (화면 잘림, 스크롤, 성능 테스트)
- 전자칠판 화면 검수 (전체화면, 터치 조작 확인)
- 수업용 링크/QR 공유 흐름 실제 테스트

### P1. 단기 개선
- 이미지 용량 추가 최적화 (모바일 데이터 환경 대비)
- 학생용 첫 화면 문구 단순화 (직관적인 가이드 제공)
- 포켓/몬스터볼 계열 표현 자체 용어화 (저작권 고려하여 '탐험 도구' 등의 용어로 변경)

### P2. 선택 개선
- 구형 iPad `inert` 폴리필 적용 (접근성 문제 해결)
- 교사용 추천 퀴즈 세트 제공
- 수업 후 정리 화면 추가 구성

## 4. 하지 않을 일
- 서버 사이드 렌더링(SSR) 적용 (정적 사이트로 유지)
- 사용자별 클라우드 로그인 및 동기화 (학생 개인정보 보호를 위해 LocalStorage 유지)
- 과도하게 복잡한 애니메이션 추가 (저사양 기기 성능 저하 방지)
